# AI Toolkit

A curated repository of proven AI configurations, commands, workflows, and code patterns for solo developers.

## Project Structure

```
ai-toolkit/
  api/                    # Programmatic AI usage (SDKs, scripts)
    claude/               # Anthropic Claude API patterns
    gemini/               # Google Gemini API patterns
  end-user/               # Power user configs (no coding required)
    claude/               # Claude Code tooling
      commands/           # Slash commands (e.g., solo-dev-workflow)
      hooks/              # Hook examples
      mcp-servers/        # MCP server configs
      settings/           # Settings examples (CLAUDE.md templates, etc.)
      skills/             # Skill definitions
    gemini/               # Gemini configs (future)
  docs/                   # Project documentation
    PRD.md                # Product requirements
    backlog.md            # Ideas and future work
```

## What Belongs Here

- Proven, reusable prompts and commands
- Tool configurations that work
- Code patterns you'd actually copy into projects
- Clear documentation for immediate use

## What Does NOT Belong Here

- Experimental stuff (goes to ai-playground)
- Project-specific code
- Half-baked ideas or "sort of works" solutions
- Tutorials or learning content

## Quality Standards

- **High bar for inclusion** - Only add things that are proven and polished
- **Organized and accessible** - Easy to find, easy to use
- **Clear and crisp** - Every item should be immediately understandable
- **Stay relevant** - Remove or update things that become outdated
